# 🔮 Out-of-Office Oracle

Generate dramatically over-the-top out-of-office messages powered by Claude AI.

## Setup (takes 2 minutes)

**1. Install dependencies**
```bash
npm install
```

**2. Add your API key**
```bash
cp .env.example .env
# Then open .env and paste your key from https://console.anthropic.com/
```

**3. Run the app**
```bash
npm run dev
```

Open http://localhost:5173 — the oracle awaits.
